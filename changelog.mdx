## Upgrading to the Latest Version

Follow these steps to upgrade Harsh Savani. Always back up your data first.

<Steps>
  <Step title="Check Current Version" icon="package">

    Run the version check command:

````bash
npx harsh-savani --version
````

  </Step>
  <Step title="Update Package" icon="download">

    <Tabs>
      <Tab title="npm" icon="npm">
````bash
npm install harsh-savani@latest
````
      </Tab>
      <Tab title="yarn" icon="yarn">
````bash
yarn add harsh-savani@latest
````
      </Tab>
    </Tabs>

  </Step>
  <Step title="Restart Server" icon="refresh-cw">

    Restart your development server:

````bash
npm run dev
````

  </Step>
</Steps>

<Callout kind="tip">
After upgrading, review the [migration guide](https://github.com/example/harsh-savani/blob/main/MIGRATION.md) for any breaking changes.
</Callout>
